Inspiring Moments

A collection of time-constrained Grain video segments using multiple URL parameters

This page demonstrates how to embed Grain videos with precise time constraints using multiple URL parameters for maximum compatibility. Each segment below uses a combination of start, end, t, time, and URL fragment #t= parameters to ensure the video plays only the specified portion.

Implementation Notes

This page uses the custom Grain player wrapper to enforce time constraints:

const player = new GrainPlayerController({
    containerId: 'player-container-1',
    recordingId: 'RECORDING_ID',
    accessToken: 'ACCESS_TOKEN',
    startTime: 120,
    endTime: 165,
    autoplay: false,
    onReady: () => { console.log('Player is ready'); },
    onTimeUpdate: (time) => { console.log('Current time:', time); },
    onEnd: () => { console.log('Segment ended'); }
});

This approach provides more reliable control over video segments by using multiple techniques to enforce time constraints.
