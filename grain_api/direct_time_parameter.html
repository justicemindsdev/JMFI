Grain Direct Time Parameter Example

This example demonstrates how to use the t parameter to set the start time for a Grain video embed.

Note: The t parameter sets the start time in seconds. In this example, the video will start at 2 minutes and 50 seconds (170 seconds).

Example Video (Starting at 2:50)

HTML Code

<iframe 
    src="https://grain.com/_/embed/recording/01452015-60e4-42a6-9ca7-f241f18ee7db/oMarnTp5zhCtrRsJJ1Sw1t1t4Q9A2qcMjpSMvoz6?autoplay=true&t=170" 
    allow="fullscreen" 
    allowfullscreen="true" 
    frameborder="0">
</iframe>

Using with timeStringToSeconds

If you're using a template system or JavaScript to generate the iframe, you can convert time strings to seconds:

// Convert time string to seconds
function timeStringToSeconds(timeString) {
    const parts = timeString.split(':');
    let seconds = 0;
    
    if (parts.length === 3) {
        // Hours:Minutes:Seconds format
        seconds = parseInt(parts[0]) * 3600 + parseInt(parts[1]) * 60 + parseInt(parts[2]);
    } else if (parts.length === 2) {
        // Minutes:Seconds format
        seconds = parseInt(parts[0]) * 60 + parseInt(parts[1]);
    }
    
    return seconds;
}

// Example usage
const startTimeString = "00:02:50";
const startTimeSeconds = timeStringToSeconds(startTimeString);

// Generate iframe with t parameter
const iframe = `<iframe 
    src="https://grain.com/_/embed/recording/01452015-60e4-42a6-9ca7-f241f18ee7db/oMarnTp5zhCtrRsJJ1Sw1t1t4Q9A2qcMjpSMvoz6?autoplay=true&t=${startTimeSeconds}" 
    allow="fullscreen" 
    allowfullscreen="true" 
    frameborder="0">
</iframe>`;
Important: While the t parameter works for setting the start time, controlling the end time requires JavaScript as shown in the Time Range Player example.
